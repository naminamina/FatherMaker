# Father Maker

**デモURL**  
https://fathermaker.onrender.com/

## 概要
Father Maker は、性格診断から父アバターを生成し、その父がWeb上で日中を過ごす様子を眺められるアプリです。

ターゲットは、日中、父がどのように過ごしているのかわからない子です。  
仕事中の様子、休憩のしかた、疲れたときの感じなど、家族の前では見えにくい父の時間を、アバターを通して想像できるようにしました。

## 体験
ユーザーは短い診断に答えることで、うちの父っぽい属性を持ったアバターを生成します。  
生成された父は、仕事をしたり、休憩したり、移動したり、考えごとをしたりしながら、アプリの中で時間を過ごします。
の様子を見たり、短いセリフやログを読んだりすることで、「いま本物のお父さんは何をしているのだろう」「アプリ父の行動を父に共有してみよう」
といったコミュニケーションのきっかけが自然に生まれることを目指しています。

直接話すのは少し照れくさくても、まずはアバターをきっかけに父を創造しそこから、会話やメッセージのきっかけが生まれる体験を提案しています。

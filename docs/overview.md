# Father Maker 概要

## 背景
日本の父親像は「仕事中心で家では寡黙」というイメージが強く、家族がその見えない時間を知る機会は少ないままです。Father Maker はデジタルの場に父の生活断片を持ち込み、日々の当たり前を見つめ直すきっかけを提供します。

## なぜ父の日なのか
父の日は「感謝を伝えたいのにきっかけが無い」日に位置づけられます。本プロジェクトは贈り物ではなく、父を想像する時間そのものを贈ることを目的にしています。診断→観察という軽い体験で、父の日当日だけでなく準備期間から話題にできます。

## 何が新しいのか
- キャラクター生成を「父らしさ」の再確認に使う点
- 状態遷移やログを通じて、見えない時間の積み重ねを感じさせる演出
- AI テキスト生成とフォールバック文を組み合わせ、通信状況に左右されない穏やかな体験

## ユーザー体験の流れ
1. **イントロ**: コンセプト説明と「父を見守る」意図を伝える。
2. **簡易診断**: 数問の設問から 5 アーキタイプのいずれかを決定。
3. **結果画面**: タイプ名・タグ・説明を表示し、父との向き合い方を提示。
4. **メイン画面**: 父キャラクターが状態変化し、セリフ・ログ・ステータスが更新される。
5. **サブパネル**: ログ確認、アイテム使用、1 日のまとめを閲覧。
6. **リロード後**: localStorage により途中状態から再開でき、継続的に父を観察できる。
